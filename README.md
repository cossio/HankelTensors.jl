Hankel tensors in Julia.

## Installation

This package is not registered.
Install with:

```julia
import Pkg
Pkg.add(url="https://github.com/cossio/HankelTensors.jl")
```

This package does not export any symbols.

## Compat

Needs Julia 1.7, because type inference fails sometimes on Julia 1.6.
